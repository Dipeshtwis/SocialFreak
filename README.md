## Getting Started <a name = "gs"></a>

To get a local copy of the repository please run the following commands on your terminal:

```
$ cd <folder>
```

~~~bash
$ git clone git@github.com:Dipeshtwis/SocialFreak.git
$ cd SocialFreak
$ bundle install
$ yarn install --check-files
$ rails db:migrate
~~~
